Gitee + PicGo实现博客图床

2022-01-31

前置准备：

  • Gitee注册账号Gitee - 基于 Git 的代码托管和研发协作平台
  • PicGo下载安装Releases · Molunerfinn/PicGo · GitHub
  • Typora下载安装https://typora.com.cn/

操作步骤

  • Gitee
    1. 新建一个仓库
    2. 在个人设置里面找到私人令牌
    3. 点击生成新令牌
    4. 勾选projects选项
    5. 生成令牌
    6. 输入账户密码
    7. 得到token（自行保存）
  • PicGo
    1. 安装插件
    2. 设置图床
  • Typora
    1. 修改偏好设置
    2. 验证上传
    3. 浏览器访问成功

Gitee

1. 创建一个仓库

2. 获取token

2.1 打开设置

2.2 找到私人令牌

2.3 生成新令牌

2.4 输入密码

2.5 获取到token并自行保存

自此，Gitee操作完毕，要注意保存好自己的token，在页面上也提示了，当你确认并关闭之后，将获取不到了（如果丢失，重新生成即可）

PicGo

打开PicGo，在插件位置搜索gitee，将搜索出来的插件全部安装（如果安装失败，可以尝试安装node环境之后再安装）

配置图床

Typora

ctrl + 逗号打开偏好设置

点击验证图片上传选项

将得到的url地址粘贴至浏览器访问。

大功告成！！！
